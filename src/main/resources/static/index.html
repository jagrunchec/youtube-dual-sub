// FONCTIONNALITÉS DU LECTEUR

⏩
Vitesse de lecture

Boutons 0.75× · 1× · 1.25× · 1.5× · 2× dans la barre sous la vidéo. Raccourcis clavier + / -.

≡
Transcription complète

Bouton TRANSCRIPTION (raccourci T). Toutes les lignes défilent en temps réel. Cliquez sur n'importe quelle ligne pour y sauter directement.

⟳
Boucle de phrase

Cliquez le bouton ⟳ à droite de n'importe quelle ligne pour la répéter indéfiniment. Recliquez ou appuyez L pour stopper la boucle.

🔍
Recherche dans la transcription

Saisissez un mot dans la barre de recherche du panneau transcription. Naviguez entre les résultats avec les boutons ▲ ▼.

↓
Export SRT / TXT

Téléchargez les sous-titres au format .srt (compatible avec VLC, MPC-HC…) ou .txt interleaved avec les deux pistes entrelacées.

⏸
Auto-Pause

Bouton ⏸ AUTO-PAUSE dans la barre du lecteur. La vidéo se met automatiquement en pause à la fin de chaque phrase. Une barre apparaît sous les sous-titres — cliquez dessus ou appuyez Espace pour passer à la phrase suivante. Idéal pour les apprenants de niveau intermédiaire qui souhaitent écouter, comprendre et répéter chaque phrase à leur rythme.

🔗
Copier l'URL

Bouton 🔗 URL dans la barre du lecteur. Copie l'URL YouTube de la vidéo en cours dans le presse-papier.

▶
Mémorisation de position

La position est sauvegardée automatiquement toutes les 5 secondes. La prochaine fois que vous ouvrez la même vidéo, un bandeau vous propose de reprendre là où vous étiez.

// FONCTIONNALITÉS GÉNÉRALES

🌐
Mode Immersion

Cochez Mode Immersion sur l'écran d'accueil. La piste 1 affiche la langue originale de la vidéo (sans traduction), la piste 2 est traduite dans votre langue. Idéal pour l'apprentissage naturel.

⊕
Bookmarklet navigateur

Glissez le lien ⊕ DualSub (visible sur la page d'accueil) dans votre barre de favoris. Cliquez-le depuis n'importe quelle page YouTube pour l'ouvrir automatiquement dans DualSub.

☀
Thème clair / sombre

Bouton ☀ THÈME dans la barre en haut à droite. Votre choix est mémorisé entre les sessions.

📊
Statistiques personnelles

Ouvrez votre Profil pour consulter le nombre de vidéos vues, votre activité des 8 dernières semaines et vos paires de langues favorites.
